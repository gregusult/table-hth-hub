# table-hth-hub

## 项目简介

`table-hth-hub` 是一个用于归档和发布独立 HTML 页面的仓库。这里存放的是多个彼此无关、不针对特定网站或域名的静态页面文件，主要目的是提供一种集中存储和版本管理的方式。

这些页面可能包括工具页面、信息展示页、实验性页面等，各自独立运行，不依赖外部服务或后端。

## 目录说明

仓库根目录下直接存放各个 HTML 文件，每个文件为一个独立页面。目录结构如下：

```
/
├── page1.html
├── page2.html
├── ...
└── README.md
```

- 每个 HTML 文件均自包含，无跨文件依赖。
- 文件名应尽量简洁，反映页面内容或用途。
- 如有子目录，会在对应位置说明。

## 页面归档说明

- 所有页面均为静态 HTML，可离线查看或直接通过 GitHub Pages 等静态托管服务访问。
- 页面内容不包含任何跟踪脚本、广告或外部资源（除非必要且明确标注）。
- 归档目的仅为保存与分享，不鼓励也不反对任何特定使用方式。
- 每个页面的维护状态可能不同，部分页面可能长期未更新但仍保留作为参考。

## 维护说明

- 本仓库由维护者不定期更新，不承诺固定发布周期。
- 欢迎提交 Issue 或 Pull Request 来报告问题、提出改进或新增页面。
- 提交的页面应遵守以下基本要求：
  - 无恶意代码、无侵犯他人权益的内容。
  - 尽可能保持文件自包含，避免引入大量外部依赖。
  - 文件名避免特殊字符，使用小写字母和连词线。
- 维护者有权对提交内容进行审查、修改或拒绝。
